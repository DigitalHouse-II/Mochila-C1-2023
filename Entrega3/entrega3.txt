¿ Por qué el lenguaje de programación sólo se utiliza en algunos sistemas operativos y en otros no? 
Los lenguajes de programacion pueden estar limitados a ciertos sistemas operativos debido a las diferencias en sus bibliotecas y api de sistema disponibles en cada sistema operativo. 
Algunos lenguajes de programación pueden estar asociados con un sistema operativo en particular debido a su historial de desarrollo y uso.

¿Qué tipo de máquina virtual es virtualBox?
Es un tipo de software de virtualización que se conoce como "hipervisor de tipo 2" o "hipervisor de host". Esto significa que se ejecuta en un sistema operativo host (como Windows, macOS o Linux) permitiendo crear y ejecutar múltiples máquinas virtuales aisladas en las que se pueden instalar diferentes sistemas operativos invitados. VirtualBox crea una capa adicional entre el hardware físico de la computadora y los sistemas operativos invitados, lo que permite que varias máquinas virtuales se ejecuten en una sola computadora física.

¿esto afecta a las demás? ¿por qué?
La falla o rotura de una máquina virtual no afecta directamente a otras máquinas virtuales que se estén ejecutando en el mismo hipervisor, ya que cada máquina virtual se ejecuta en su propio entorno aislado. Pero es posible que el rendimiento del hipervisor se vea afectado por el fallo, lo que puede afectar a todas las máquinas virtuales que se están ejecutando en el mismo host.
