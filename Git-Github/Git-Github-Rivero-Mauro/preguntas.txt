¿Quién inventó el sistema de control de versión Git y por qué?
El sistema de control de versiones Git fue creado por Linus Torvalds.
Creo Git como solución a otros sistemas de control de versiones que eran más lentos y limitados a proyectos no muy grandes.


¿A quién pertenece actualmente GitHub y por qué?
Actualmente GitHub pertenece a Microsoft, el cual fue comprado por 7500 millones de dólares.
La compra se debe a una estrategia de Microsoft para tener más presencia en el mercado del desarrollo de software código abierto.

¿Hay otra forma que no sea la terminal para trabajar con GitHub?

Si, existen varias formas:

GitHub Desktop 
Web de GitHub
Extensiones de Git para editores de código
Integraciones en otras herramientas
