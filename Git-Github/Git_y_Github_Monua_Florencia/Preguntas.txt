¿Quién inventó el sistema de control de versión Git y por qué?
Linus Torvalds. 
En el 2005, la relación entre la comunidad que desarrolló el kernel de Linux y la compañía que desarrolló BitKeeper se vino abajo y la herramienta dejó de ser ofrecido de manera gratuita. Esto impulsó a la comunidad de desarrollo de Linux (y en particular a Linus Torvalds, el creador de Linux) a desarrollar su propia herramienta basada en algunas de las lecciones que aprendieron mientras usaban BitKeeper.
Fuente: https://git-scm.com/book/es/v2/Inicio---Sobre-el-Control-de-Versiones-Una-breve-historia-de-Git
